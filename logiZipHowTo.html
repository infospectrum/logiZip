About logiZip

logiZip is a free Windows tool that helps users deploy via the manifest controlled Module Loader of SugarCRM:

  • Logic Hooks
  • Scheduler Tasks
  • Custom Directory Changes
  • Custom Fields in Modules

logiZip automates the manifest.php file content creation and zips it along with the source files in a package, ready for uploading to SugarCRM on-demand or on-premise instances.

Using Module Loader enables administrators to easily enable or disable logic hooks and other modules. logiZip saves the deployment parameters for each generated manifest in its own configuration file, making them available for future updates.

Prerequisites

PShellExec utility (included and free)
PowerShell 2.0

It runs under the following Windows Systems:

- Windows XP and Vista
- Windows 7 and 8
- Windows Server 2003 and 2008

Installation

[1] Please make sure the prerequisites listed above are installed first. First time PowerShell users may also need to set authorizations from the command prompt:

  • type PowerShell
  • at the PS prompt type Set-ExecutionPolicy RemoteSigned
  • type exit to return to the command prompt and close the command prompt window.

[2] logiZip is distributed as logiZip.zip file. Unzip it in a <directory> and then run the setup.cmd.


Running logiZip

Click the logiZip shortcut in your <directory> to start it.

Select a new or a previously saved configuration. Select or drag-and-drop as many php source code files you would like to place in one package. logiZip configures the manifest content depending on the type of selected source files.

Right-click on a Php Sources file and then select: Open to view it in the Source Code tab, or Setup to select a logic hook type or setup the Scheduler job parameters.

Use the Manifest button to review and edit the formatted entries in the Manifest tab. Click the Zip button to create a .zip package an save the configuration file.

SugarCRM Login is optional.The login to a Sugar instance lets users select from the list of modules, and shows the fields for a selected module, as well as the relationships defined for the selected module (including custom modules).

Search modules by selecting [Right-click] module fields to be listed, and by entering the search values. Users can also edit the search query [using for AND/NOT/OR logic] in the Search tab before displaying the results.

Use the generated .zip package as input to SugarCRM's Module Loader to finish your installation.

Logic Hooks

Several Logic Hooks sources can be included under one package name. Note that each Logic Hook source can include multiple functions. In such cases, before zipping the package, care should be taken to update manually the hook 'type' in the manifest tab accordingly.

Scheduler Tasks

For custom Scheduler tasks logiZip generates a language definition file with $mod_strings['LBL_SOMENAME'] = 'Some Job Entry Name' matching the $job_strings[] = 'SomeName' from the selected source. Module Loader automatically takes care of propagating the Scheduler Task so that Some Job Entry Name name will appear in the dropdown of the job Scheduler. In addition, users can enter the job run parameters for a scheduled task or include their own /scripts/post_install.php source.

Custom Directory Changes

The /custom/ directory is the place for upgrade-safe customizations. In on the on-demand environments the directory is accessible through Module Loader in write-only mode. Check the Custom Copy in the Setup Tab and select the files from your local /custom/<directory>/ tree. logiZip packages your customizations for deployment and control via Module Loader. On-demand users can download their /custom/ directory via Admin > System > Diagnostic Tool > SugarCRM Custom directory selection before or after their customizations.
